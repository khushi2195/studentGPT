# 🎓 StudentGPT

StudentGPT is an AI-powered web application designed to help students with their studies by providing instant answers, explanations, 
and guidance on various topics.It acts like a smart assistant for learning.

Features:-
Ask questions and get instant answers
Helps with coding, concepts, and general queries
Simple and user-friendly interface
Fast response using AI integration

Technology used in project:
Frontend: React.js
Backend: Node.js, Express.js
API: OpenAI / AI model integration
Database: MongoDB (if used)

How It Works:-
User enters a question
Request is sent to backend
Backend calls AI API
AI generates response
Answer is shown to the user

Future Improvements:-
Voice input feature
Personalized learning suggestions
